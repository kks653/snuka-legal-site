Report a Generated Print

Use the in-app Report generated print action on the affected output. Select a safety reason and optionally add up to 1,000 characters of context. This sends a private request to Snuka operations for review; it does not automatically remove, block, classify, or regenerate the print. For urgent support that cannot be submitted in the app, email the support address above.

Deletion Controls

The pack deletion control removes only the current photo session and its private image files. The separate account deletion control removes the anonymous account and user-facing Supabase data; a temporary non-image signed-upload security tombstone remains until its post-expiry storage cleanup succeeds. See Delete Data for the exact scope and retained-record limits.
